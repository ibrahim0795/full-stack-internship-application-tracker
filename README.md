# CareerOrbit

CareerOrbit is a full-stack internship and junior-job application tracker. It will help users organise opportunities, monitor application progress, prepare for interviews, manage CV versions, and review application analytics.

## Project status

Repository initialized. Development will proceed incrementally through focused feature branches.

## Planned stack

- Next.js, React, and TypeScript
- Tailwind CSS
- PostgreSQL and Prisma
- Auth.js
- React Three Fiber, Drei, and GSAP
- Vitest and Playwright
